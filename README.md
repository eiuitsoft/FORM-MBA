# FORM-MBA
Form nhập liệu MBA - Eastern International University

## Mô tả
Form đăng ký chương trình MBA (Master of Business Administration) cho Eastern International University.

## Công nghệ sử dụng
- Angular 21
- TypeScript 5.9
- Tailwind CSS
- Reactive Forms

## Cài đặt

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Build production:
   ```bash
   npm run build
   ```
